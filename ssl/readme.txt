 openssl req -x509 -nodes -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365

 Эта команда `openssl req -x509 -nodes -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365`
 генерирует самоподписанный SSL-сертификат (cert.pem) и закрытый ключ (key.pem) с использованием
 алгоритма RSA на 4096 бит. Опция -x509 указывает на создание самоподписанного сертификата, -nodes отключает
 шифрование ключа с помощью фразы пароля, -newkey rsa:4096 создает новый ключ RSA на 4096 бит, -keyout указывает
 имя файла для закрытого ключа, -out указывает имя файла для сертификата, -days указывает срок действия сертификата
 в днях.
